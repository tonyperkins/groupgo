Session A — Theaters page addition

Venue — Showtime URL pattern field

Added to the Create/Edit Venue form. Set once per theater. Powers "↗ verify" links on the Times page.

(optional)

Used to generate "↗ verify" links next to each fetch date. Use {date} where the date should appear (format: YYYY-MM-DD).

Preview
Sat Mar 21 → https://www.cinemark.com/theatre/cinemark-cedar-park?showDate=2026-03-21
Sun Mar 22 → https://www.cinemark.com/theatre/cinemark-cedar-park?showDate=2026-03-22
Common patterns
Cinemark: …?showDate={date}
AMC: …/showtimes?date={date}
Regal: …?date={date}&theatreId=…
How verify links render on Times page
Date pills with verify links
Sat, Mar 21
↗ verify AMC ↗ verify Cinemark
Sun, Mar 22
↗ verify AMC ↗ verify Cinemark

Each "↗ verify" link opens the theater's showtime page for that specific date, using the showtime_url_pattern with {date} replaced. If no pattern is set, links to the theater's homepage. If neither, no link shown.
